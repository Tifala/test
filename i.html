Chapitre 2: Accédez au terminal

Qu'est ce que le terminal ?

Le terminal est un application qui vous permet d'interagir avec votre ordinateur à travers des commandes (vous donnez des instructions à votre ordinateur et il l'exécute pour vous). Le terminal vous permet de faire la majorité des actions faites avec l'interface utilisateur et plus encore. Lorsque vous utilisez le terminal c'est plus direct, plus rapide, et pour un développeur l'optimisattion de son temps est très important.

Comment accéder au terminal

Cela dépend de votre système d'exploitation. Ici nous utilisons le MacOs et pour accéder au terminal, nous avons plusieurs méthodes:

  1. Ouvrir le dossier Applications > Utilities > Terminal
  2. Appui simultané sur la touche Command et la barre d'espace > dans la barre de recherche tapez Terminal

Vous pouvez épinglez le Terminal afin de l'avoir toujours dans vos logiciels favoris

Exemple de ligne de commande dans un terminal

  1. Chaque ligne de command commence par le prompt et généralement avec le signe dollars $. Ceci vient automatiquement selon la configuration de votre machine et ce n'est pas à vous de taper le prompt
  2. Ensuite nous avons la commande mkdir. Ce dernier indique l'action que nous voulons exécuter. Dans notre cas la commande mkdir (qui signifie make directory) veut dire créer un dossier
  3. Pour terminer, nous avons l'argument. Notre argument ici c'est cours_cli qui a pour action mkdir.

Donc en résumé notre ligne de commande mkdir cli_course veut dire créer le dossier cours_cli
